# Installation

## Requirements

- Zig 0.15.1 or later

## Build from Source

```bash
git clone https://github.com/brian-sinquin/mimg.git
cd mimg
zig build
```

The binary will be in `zig-out/bin/mimg`.

## Development

```bash
# Run tests
zig build test

# Generate example gallery
zig build gallery
```</content>
